celect - Multi Select

How to initialize?

Initialize in document ready function as below:

$(document).ready(function(){
$("#drpcar").celect({multi:true,title:'Select Cars'});
$("#drpbike").celect({multi:true,searchable:true,title:'Select Bikes'});
});

If you want to bind click, change or any other event, update value, that can be done as below:

$(document).ready(function(){
$("#drpcar").celect({multi:true,title:'Select Cars'});
$("#drpbike").celect({multi:true,searchable:true,title:'Select Bikes'});
$("#drpcar").on("change",function(){alert('updated');});
$("#drpbike").on("change",function(){alert('updated');});
$("#drpcar").celectUpdate([2,5,7,9]);
$("#drpbike").celectUpdate([1,3,5]);
});
Get the selected values by calling given method:

$("#drpcar").celectGet();
