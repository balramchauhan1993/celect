Custom Select

How to initialize?

Initialize in document ready function as below

$(document).ready(function(){
$("#drpcar").celect();
$("#drpbike").celect();
});

If you want to bind click, change or any other function then bind after initialization, like given below:

$(document).ready(function(){
$("#drpcar").celect();
$("#drpbike").celect();
$("#drpcar").on("change",function(){alert('updated');});
$("#drpbike").on("change",function(){alert('updated');});
$("#drpcar").celectUpdate(5);
$("#drpbike").celectUpdate(5);
});
