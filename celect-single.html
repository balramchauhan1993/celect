celect - Single Select

How to initialize?

Initialize in document ready function as below:

$(document).ready(function(){
$("#drpcar").celect();
$("#drpbike").celect();
});

If you want to bind click, change or any other event, update value, that can be done as below:

$(document).ready(function(){
$("#drpcar").celect();
$("#drpbike").celect();
$("#drpcar").on("change",function(){alert('updated');});
$("#drpbike").on("change",function(){alert('updated');});
$("#drpcar").celectUpdate(5);
$("#drpbike").celectUpdate(5);
});
To get the selected value:

$("#drpcar").celectGet();
$("#drpbike").celectGet();

or (below method is only for single select)

$("#drpcar").val();
$("#drpbike").val();
